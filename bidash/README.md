# bidash

損益計算書・貸借対照表のCSVから、インタラクティブなHTMLダッシュボードを生成するRパッケージです。

## インストール

```r
# devtools が必要です
install.packages("devtools")
devtools::install_github("your-username/bidash")
```

## 使い方

```r
library(bidash)

# 1. テンプレートCSVをカレントディレクトリにコピー
use_templates()

# 2. CSVを自社データに編集する（フォーマットは下記参照）

# 3. ダッシュボードを生成
create_dashboard(company_name = "株式会社サンプル (1234)")
```

サンプルデータで試したい場合：

```r
library(bidash)

use_examples()  # 架空のサンプルデータをコピー
create_dashboard(company_name = "株式会社サンプルテック (9999)")
```

## 生成されるダッシュボード

| タブ | 内容 |
|---|---|
| 業績サマリー | KPIボックス・売上利益トレンド・ROE推移（5期分） |
| 損益計算書 | ウォーターフォール・利益率比較・費用構造・詳細テーブル |
| 貸借対照表 | 資産/負債構成・財務健全性指標・詳細テーブル |

## CSVフォーマット

3つのCSVファイルを用意します。文字コードは **UTF-8（BOM付き）** です。

---

### `input_pl.csv` — 損益計算書

- 1列目：`項目`（固定）
- 2列目以降：期名（列名が自動的に期ラベルとして使われます）
- **直近2列**が「前期」「当期」として比較に使われます

```
項目,2024年3月期,2025年3月期
売上高,4920000,5340000
売上原価,2460000,2670000
売上総利益,2460000,2670000
販売費及び一般管理費,2050000,2220000
営業利益,410000,450000
営業外収益合計,18000,22000
営業外費用合計,18000,22000
経常利益,410000,450000
特別利益合計,5000,8000
特別損失合計,5000,8000
税金等調整前当期純利益,410000,450000
法人税等合計,125000,140000
当期純利益,285000,310000
```

> 単位：千円。`項目` 列の名称はダッシュボードのロジックで参照されるため**変更不可**です。

### `input_bs.csv` — 貸借対照表

- 1列目：`区分`（`資産` / `負債` / `純資産` のいずれか）
- 2列目：`項目`（固定）
- 3列目以降：期名

資産,資産合計,2980000,3250000
負債,流動負債合計,780000,820000

### `input_summary.csv` — 複数期サマリー

- 行が期（古い順に並べてください）
- 列名は固定です

```
期,決算年月,売上高,経常利益,当期純利益,純資産,総資産,自己資本比率,ROE,EPS
第1期,2021/3,3850000,280000,190000,1200000,2100000,55.2,16.8,38.50
第2期,2022/3,4120000,320000,215000,1380000,2350000,57.4,16.8,43.60
第3期,2023/3,4580000,380000,260000,1590000,2620000,59.1,17.8,52.70
第4期,2024/3,4920000,410000,285000,1820000,2980000,60.3,16.8,57.80
第5期,2025/3,5340000,450000,310000,2070000,3250000,62.5,15.8,62.90
```

| 列名 | 内容 | 単位 |
|---|---|---|
| `期` | 期の名称（表示用） | — |
| `決算年月` | X軸ラベル（例：`2025/3`） | — |
| `売上高` | 売上高 | 千円 |
| `経常利益` | 経常利益 | 千円 |
| `当期純利益` | 当期純利益 | 千円 |
| `純資産` | 純資産合計 | 千円 |
| `総資産` | 資産合計 | 千円 |
| `自己資本比率` | 純資産／総資産 × 100 | % |
| `ROE` | 当期純利益／純資産 × 100 | % |
| `EPS` | 1株当たり当期純利益 | 円 |

> 期数は3期以上あると推移グラフが見やすくなります。

## 関数リファレンス

### `create_dashboard()`

```r
create_dashboard(
  pl_file      = "input_pl.csv",       # 損益計算書CSVのパス
  bs_file      = "input_bs.csv",       # 貸借対照表CSVのパス
  summary_file = "input_summary.csv",  # 複数期サマリーCSVのパス
  company_name = "財務分析",           # ダッシュボードのタイトルに表示
  output_file  = "dashboard.html",     # 出力HTMLファイルのパス
  open         = TRUE                  # 生成後にブラウザで開くか
)
```

### `use_templates()`

空のテンプレートCSV（自社データ入力用）をコピーします。

```r
use_templates(dir = ".", overwrite = FALSE)
```

### `use_examples()`

サンプルデータ（架空の「株式会社サンプルテック」）をコピーします。

```r
use_examples(dir = ".", overwrite = FALSE)
```
